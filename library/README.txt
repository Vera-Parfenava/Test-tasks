# Библиотечное приложение

Это простое приложение для управления библиотекой. Оно позволяет добавлять книги, удалять их, изменять их статус, а также искать книги по различным критериям.

## Функции

- **Добавить книгу**: добавляет книгу в библиотеку.
- **Удалить книгу**: удаляет книгу по ID.
- **Поиск книги**: ищет книги по названию, автору или году издания.
- **Отобразить все книги**: выводит все книги в библиотеке.
- **Изменить статус книги**: изменяет статус книги на "в наличии" или "выдана".

## Стек технологий

- Python 3.12.5
- JSON для хранения данных
- Модуль `typing` для аннотирования типов

## Запуск приложения

1. Клонируйте репозиторий.
2. Запустите файл `main.py`.
3. Следуйте инструкциям на экране для взаимодействия с приложением.
